# 📚 书香中国电子书下载与去DRM解密合并工具

本工具专为下载并解密 **书香中国（ChineseAll）** 平台的 PDF 电子书而设计。它能够自动捕获动态派生的解密密钥、高速并发下载单页、并在本地自动实现 PDF 密码解密（去除 DRM 限制），最终按页码顺序合并输出为无限制的高清离线 PDF 电子书。

---

## 🛠️ 环境依赖

在运行前，请确保您的系统已安装以下环境：
1. **Python 3.7+**
2. **相关依赖库**：
   在命令行中执行以下命令安装：
   ```bash
   pip install httpx playwright pypdf cryptography
   playwright install chromium
   ```

---

## 🚀 使用方法

### 一键下载当前书籍
1. 双击打开文件夹中的 **`运行下载器.bat`**。
2. 此时会在您的桌面上弹出一个 Chromium 浏览器窗口，并自动载入目标书籍页面。
3. **根据您的阅读权限决定是否登录**：
   * **若您需要完整版（整本书）**：请在弹出的浏览器中**登录您的读者证/机构账号**。登录成功并进入阅读器后，脚本将自动感应到完整阅读权限，并立刻开始全自动下载！
   * **若您只需要前 10 页（试读版）**：无需任何操作，静候约 10~15 秒，脚本检测完毕后会自动关闭浏览器并下载前 10 页。
4. 页面关闭后，脚本会在后台自动完成：
   `探测页数 ➔ 并发下载加密单页 ➔ 本地密码解密 ➔ 页面按序合并`。
5. 最终生成的无密码 PDF 文件将保存在本目录下，命名为 `book_xxxx.pdf`。

## 📝 下载其他书籍

如果您需要下载平台上的其他电子书，只需两步：
1. 复制该电子书在浏览器地址栏中的链接。例如：
   `https://shuxiang.chineseall.cn/book/read/FjdEj`
   链接最末尾的 **`FjdEj`** 就是该书的 `BOOK_ID`。
2. 用文本编辑器打开本目录下的 **`download.py`**，修改第 **9** 行的配置项：
   ```python
   # 如果想下载其他书籍，请修改这里的 BOOK_ID
   BOOK_ID = "您的新图书ID"
   ```
3. 保存并重新运行 `运行下载器.bat` 即可！
